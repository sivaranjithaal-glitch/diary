☁ How Sync Works
This app uses JSONBin.io — a free cloud storage service. Your case data is saved to a private bin only you can access. When you update a case on your Windows PC, the data automatically uploads. When you open the app on your Android phone (with internet), it pulls the latest data.
Free tier: 10,000 requests/month · Data encrypted in transit · Your private API key protects your data
🔧 Setup (One Time)
  1. Go to jsonbin.io and create a free account
  2. Go to API Keys → copy your Master Key
  3. Paste it below and click Create New Bin
  4. Install the app on Android (same WiFi or internet) and enter the same API Key + Bin ID
From jsonbin.io → API Keys → Master Key
Copy from jsonbin.io or click "Create Bin" to auto-create
Pull vs Force Pull: Pull only updates if cloud has newer & equal/more records (safe). Force Pull replaces all local data with cloud data — use only for recovery.
